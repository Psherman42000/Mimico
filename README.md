# Mimico 🎙️🔊

Aplicativo em segundo plano para Windows que captura áudio do sistema, transcreve, traduz em tempo real e oferece tradução automática com voz clonada.

**Funcionalidades principais:**
- Captura de áudio do sistema (loopback) — Meet, Squad, Discord, qualquer app
- Transcrição e tradução (inglês → português) em overlay invisível
- Modo bandeja do sistema com miniatura que não aparece em gravação de tela
- Toggle de voz clonada: português falado → inglês com sua voz, injetado em microfone virtual
